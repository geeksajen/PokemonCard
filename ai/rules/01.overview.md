# Project Overview: PKCard TCG

## 1. Core Identity
PKCard is a local 2-player, browser-based Trading Card Game (TCG) prototype built with React and Vite. It emphasizes fluid drag-and-drop mechanics, synthesized audio, and a premium visual experience.

## 2. Architecture & State Management
- **Framework**: React + Vite.
- **State Handling**: A single centralized `gameState` object managed in `GameArena.jsx`. All state updates MUST use `structuredClone` for strict immutability.
- **Routing**: Handled simply in `App.jsx` (`lobby` vs `playing` states).
- **Data Layer**: `src/models/cards.js` serves as the local database for all cards (Pokemon, Energy, Trainers) and theme deck generation logic.

## 3. Gameplay Mechanics
- **Card Types**: 
  - Basic Pokemon (deploy to active/bench)
  - Stage 1 Evolution (drop onto matching basic Pokemon; inherits damage/energy)
  - Energy (attach to Pokemon to meet attack costs, 1 per turn)
  - Trainer Cards (drop on board/Pokemon for immediate effects)
- **Turn Flow**: Draw phase on turn start -> Attach Energy (optional) -> Play Cards -> Attack (ends turn).
- **Win Conditions**: 
  1. Take all Prize Cards by knocking out opponent Pokemon.
  2. Opponent has no Active Pokemon and no Bench Pokemon to promote (Bench Out).
  3. Opponent cannot draw a card at the start of their turn (Deck Out).
