# 進階遊戲打擊感與 UX 優化企劃書 (Advanced Game Feel Plan)

在基礎的版面配置與視覺風格 (Dark Cosmic 主題) 確立之後，為了讓本作具備「世界級 TCG」的沉浸感與爽快感，接下來的 UI/UX 優化將聚焦於 **「打擊感 (Game Feel)」** 與 **「引導直覺性 (Intuitive Flow)」**。

以下是四大進階優化核心提案：

---

## 1. 頂級閃卡反光特效 (Holographic Foil Shimmer)
**問題與目標**：實體 TCG 最迷人的要素是高罕貴度卡牌的閃卡反光。雖然目前已有拖曳傾斜 (Tilt) 的物理回饋，但缺乏反光質感。
**實作方向**：
- 結合目前計算滑鼠座標與傾斜角度的邏輯。
- 在 `Stage 1` 與 `Stage 2` 等高階卡牌的最上層，利用 CSS 覆蓋一個 `mix-blend-mode`（如 `color-dodge` 或 `overlay`）的雷射漸層塗層。
- 當卡牌傾斜時，動態改變 `background-position`，製造出雷射光暈或鑽石反光隨著視角流動的逼真 3D 閃卡質感。

## 2. 智慧出牌發光提示 (Playability Cues)
**問題與目標**：新手玩家常常需要花時間點擊或詳讀卡牌才知道「現在這回合我能打出什麼牌」，缺乏直覺引導。
**實作方向**：
- 系統在回合開始時掃描手牌。
- 如果某張手牌 **「滿足立刻打出的條件」**（例如：是一張可以直接附加的能量卡，或是場上已有可以對應進化的基礎寶可夢）。
- 該手牌邊緣會自動亮起「微微呼吸的金色/綠色光暈」，而無法打出的手牌則稍微調低亮度。
- **效益**：大幅降低認知負擔，讓玩家能直覺做出戰術決策。

## 3. 指向性攻擊箭頭 (Targeting Arrows)
**問題與目標**：目前在使用道具或指定攻擊目標時，點擊操作缺乏物理上的打擊連貫感。
**實作方向**：
- 實作「拖曳拉線 (Drag & Target)」的 UI 系統。
- 當玩家按住攻擊招式或指定性 Trainer 卡牌時，畫面會拉出一條「發光的雷射虛線」。
- 箭頭尖端會跟著滑鼠游標移動，當懸停在合法目標上方時（如敵方寶可夢），目標會出現強烈高亮；放開滑鼠後箭頭射出，並同時觸發傷害震動特效 (`arena-shake`)。

## 4. 殘局與戰局高潮警告 (Climax Vignette Pulse)
**問題與目標**：對戰的節奏需要情緒起伏，目前的對戰環境在滿血與殘血時的氛圍沒有差異。
**實作方向**：
- 當系統偵測到某一方的「獎賞卡 (Prize Card)」只剩最後 1 張，或是主將寶可夢血量低於 20% 時觸發殘局模式。
- 在整個遊戲畫面的最邊緣加上一層紅色脈動的暗角濾鏡 (`Vignette Pulse`)。
- 可配合 Web Audio API 調快背景音樂的節拍 (Tempo) 或加入心跳聲音效，無形中將對戰的緊張感推向高潮。
